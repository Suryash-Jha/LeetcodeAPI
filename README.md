# LeetcodeAPI
Get User Data using this API.

Right Now, it handles only GET request.

Just go to URL: 

https://leetapi.herokuapp.com/<user-name>

Response:

  if success:
  
  
    {
    "Status": 200,
    "User": ,
    "UserProfileLink": ,
    "Total": ,
    "Easy": ,
    "Medium": ,
    "Hard": ,
    "Rating": 
    }
    
    
  else:
  
  
    {
    "Status": 404,
    "User": 
    }
